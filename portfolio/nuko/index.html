ABOUT

NUKO is a first person multiplayer shooter which I created for my 2022 Software Development and Design major project. The game was written in C and compiled with emscripten to WASM.

For this project, I wanted to compile all the game engine technology I had learnt throughout the past years into a single game. Because of this, game was written from the ground up including physics, rendering and networking. The only libraries used was SDL2 and OpenGL.
